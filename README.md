# BPH — Biswajit Power Hub

Business management PWA for tracking inventory, sales, expenses, accounts and generating reports.

## Install as App

### Android (Chrome)
1. Open the site in Chrome
2. Tap the three-dot menu → **Add to Home screen**
3. Tap **Install**

### iPhone / iPad (Safari)
1. Open the site in Safari
2. Tap the **Share** button (box with arrow)
3. Scroll down → **Add to Home Screen**
4. Tap **Add**

### Desktop (Chrome / Edge)
1. Open the site
2. Click the install icon in the address bar (or menu → **Install BPH**)

## Features

- Inventory management with custom categories
- Buy / Sell with automatic weighted-average costing
- Capital and Profit bank accounts with full ledger
- Fixed and business expenses with category tracking
- Receivables and payables tracking
- Net Worth, P&L, Stock, Expense and Ledger reports
- PDF export via browser print
- Full offline support
- Data backup and restore (JSON)

## Data Storage

All data is stored locally in your browser's `localStorage`. Nothing is sent to any server.
Use **Settings → Export Backup** regularly to save your data.

## GitHub Pages Setup

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to **main** branch, **/ (root)** folder
4. Save — your app will be live at `https://yourusername.github.io/repo-name`
